Powerful Features

💻

Advanced Code Editor

Syntax highlighting for Kotlin, XML, JSON with auto-indent, line numbers, and AI-powered suggestions.

📁

Smart File Manager

Complete project management with GitHub integration, templates, and scoped storage support.

⚙️

Terminal Emulator

Full Linux-style terminal with Gradle commands, Git CLI, and real-time build logs.

🔨

APK Builder

Auto-detect Gradle projects, build, sign, and export APK files directly on your device.

🎨

Neon UI Design

Eye-safe dark theme with Material 3, glassmorphism, and smooth animations.

🔌

Plugin System

Extensible architecture with support for language servers and custom themes.
